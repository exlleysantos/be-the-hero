# Be-the-hero

Be The Hero é um site Web proposto na 11a Semana OmniStack, da Rocketseat.
    
## Para instalar as dependências (rodar o código em ambas as pastas) e:
```
npm install
```
## Para ativar o Backend e o Frontend

### navegar até as pastas e:
```
npm start
```
